<h3>CONCAT</h3>

```mysql
CONCAT(author_fname, ' ', author_lname) 
FROM books;

CONCAT(author_fname, ' ', author_lname)
AS 'full name'
FROM books;

SELECT 
  CONCAT_WS(' - ', title, author_fname, author_lname) 
FROM books;

```

<h3>SUBSTRING</h3>

`SELECT SUBSTRING('Hello World', 1, 4);`

`SELECT SUBSTRING('Hello World', 3);`

`SELECT SUBSTRING('Hello World', -3);`

<p>In MySQL, string index starts from 1. The above examples will result 'Hell', 'llo World', 'rld', respectively.</p>

`SELECT SUBSTRING(title, 1, 10) AS 'short title' FROM books;`

`SELECT CONCAT(SUBSTRING(title,1,10), '...') FROM books;`

<h3>REPLACE</h3>

```mysql

SELECT CONCAT(REPLACE(title, 'e', '3'),1,10)
AS 'weird string'
FROM books;

```

<h3>REVERSE</h3>
<span>Reverses a string</span>

```mysql

SELECT 
CONCAT(author_fname, REVERSE(author_fname))
FROM books;

```

<h3>DISTICT</h3>
<span>Get unique values</span>

`SELECT DISTINCT released_year FROM books;`

`SELECT DISTINCT CONCAT(author_fname, ' ', auth_lname) FROM books;`

`SELECT DISTINCT author_fname, author_lname FROM books;`

<h3>ORDER BY</h3>
<ul>
  <li>By default, it's in an ascending order. To sort by descending order, DESC must be defined at the end</li>
  <li>ORDER BY can be defined by the index number on SELECT</li>
</ul>

```mysql

SELECT title, author_fname, author_lname
FROM books
ORDER BY 3
DESC;

SELECT author_fname, author_lname
FROM books
ORDER BY author_lname, author_fname;

```

<h3>LIMIT</h3>
<ul>
  <li>LIMIT 0,3 represents row 1, and 3 is the numbe of rows. LIMIT 0, 3 will select from row 1-3</li>
  <li>LIMIT 10,1 wil only show row 11 </li>
</ul>

`SELECT title FROM books LIMIT 3;`

`SELECT title, released_year FROM books ORDER BY  released_year DESC LIMIT 0 , 3`

<h3>LIKE</h3>
<ul>
  <li>%da% - contains 'da' in any text</li>
  <li>da% - any text that starts with 'da' </li>
  <li>the- text that is exactly 'the' </li>
  <li>%the- text that ends in 'the' </li>
  <li>'_'- # of underscores repesent the number of digits</li>
</ul>

```mysql
SELECT title, author_fname 
FROM books 
WHERE author_fname 
LIKE '%da'

```
