<h3>forEach()</h3>

<blockquote>
It's a method that iterates an array and executes a provided function for each element in an array
</blockquote>

```javascript
const array = [1, 2, 3];
array.forEach(el=> console.log(el));

```

```output
1
2
3

```

<h3>splice()</h3>

<blockquote>
It's a method that changes the content of an array by REMOVING or REPLACING existing elements and/or adding new elements in place.
</blockquote>
